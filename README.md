# AuthDash - User Management Web Application

**AuthDash** is a user management web application. It provides registration, authentication, and an admin panel for managing users.

## Technologies Used:

- **Next.js 14**
- **PostgreSQL**
- **Prisma**
- **Tailwind CSS and shadcn**
